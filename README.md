# Telegram Movie Bot

A comprehensive Telegram bot for movie sharing with multi-language support, subscription management, and admin panel.

## Features

- **Multi-language support**: English, Russian, Uzbek
- **User authentication**: Phone number registration
- **Movie management**: Add, search, view movies by ID
- **Watch lists**: "Watch Later" and "Watched" sections
- **Subscription system**: Mandatory channel subscription
- **Admin panel**: User management, movie publishing
- **Database**: SQLite for data storage
